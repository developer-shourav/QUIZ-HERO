What is the difference between 'map' , 'filter', 'forEach' and 'find' array method?

The 'map()' method loop through the target array's elements and returns data in a new array format.The 'forEach()' is almost similar to 'map method. But it doesn't return anything from the function.
The 'filter()' returns an array containing the element that fulfill the condition given in the function, but 'find()' returns the element that fulfilled the condition.

What is the difference between 'LocalStorage' , 'SessionStorage' and 'Cookies'?

LocalStorage is a good way to store user browsing data of a website in his browser in string formate. The maximum capacity of LocalStorage is 10mb in browser.
SessionStorage is a great way of providing better user experience form a website for users. It's works before ending the session. If the tab or browser is closed it can't work all data will remove automatically. It maximum data storage capacity is 5 mb. Cookies are short stringified information that web application takes form user and store both browser and server for giving better experience and service. It helps website's owner / developer in many ways.
