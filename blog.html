What is the difference between 'map' , 'filter', 'forEach' and 'find' array method?

The 'map()' method loop through the target array's elements and returns data in a new array format.The 'forEach()' is almost similar to 'map method. But it doesn't return anything from the function.
The 'filter()' returns an array containing the element that fulfill the condition given in the function, but 'find()' returns the element that fulfilled the condition.
